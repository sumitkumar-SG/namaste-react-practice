Why is React known as React?
-->react is known as react because it react whenever the data changes and it automatically shows the changes on UI and name shows reactive nature of UI.When the state changes, React efficiently updates only the parts of the UI that are affected by those changes. This is achieved through a virtual DOM (Document Object Model) and a process called reconciliation, where React compares the current state with the previous one and determines the minimal set of changes needed to update the UI.
